Style Reference Guide

Purpose

The purpose of this guide is to specify the colour scheme, fonts, layout and element styles to be used in creating site pages.

Developers and designers use design systems in order to organize the look and feel of a website or web application. Design systems help to document the visual vocabulary and reusable components of a large website.

As your websites grow larger and more complex, having a consistent set of styles will help them stay cohesive and enjoyable for users! It also helps as you start working with other developers, as you can have a consistent reference for how your website should be built.

Colour Scheme

The colour scheme employed by myself is "Mohito", inspired by the colours found within the famous beverage and a nod to my favourite Linux distribution: LMDE (Linux Mint Debian Edition). Yes, I know that there are no olives in a mohito, but run with me on this, okay?

Each tone has a corresponding contrasting element colour that is also found in the same scheme.

Fonts

In yet another nod to a linux distribution, the font used in the style sheet is Ubuntu. This distribution gave me a huge amount of experience early on when it was still Debian based. An added benefit is that the font is a true type font.

The font is available from Google Fonts and can also be found in the /fonts directory of this site. The font sizes utilised are listed in the table below
